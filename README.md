# Sudoku-Solver-Visualizer

### Introduction

Play the popular japanese puzzle game sudoku and visualize the solution using the backtracking algorithm.


### Requirements
- `HTML`
- `CSS`
- `Javascript`
- `Backtracking Algorithm`
